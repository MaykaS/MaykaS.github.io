# Maya Sagalin — Personal Portfolio Website

A personal portfolio website built to showcase my background in product management, technology, and strategy.

Live site: [https://maykas.github.io/](https://maykas.github.io/)

## About

This site highlights my experience across product management, software development, project leadership, and MBA work at Cornell. It includes sections on:

- About me
- Professional experience
- Education
- Projects
- Skills
- Personal interests
- Contact information

The goal of this website is to present my work and story in a clean, structured, and personal way.

## Built With

- **Next.js**
- **TypeScript**
- **Tailwind CSS**
- **React**

## Features

- Responsive single-page portfolio layout
- Experience and education timeline
- Project showcase
- Skills section
- Personal interests section
- Contact and social links
- Deployed on **GitHub Pages**
